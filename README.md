# Lab8-Starter

Github Page: https://pacheco101.github.io/Lab8_Starter/

Short Paragraph: Service Workers serve as an ideal tool for implementing progressive enhancement and graceful degradation in modern web applications. When a user has a strong network connection, they experience the application at its highest fidelity, making live network requests for real-time data. However, if the user's connection drops or becomes highly unstable, instead of allowing the browser to fail and display a generic "No Internet" crash screen, the Service Worker intercepts the failed requests. It gracefully degrades the user experience by serving cached, local copies of the application shell and static assets. This ensures that the application remains resilient, functional, and usable under any network conditions.
